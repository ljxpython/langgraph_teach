
## 项目介绍



一个基于 langgraph 框架的教学项目

## agent_chat_ui
一个演示demo的前端页面

## 学习顺序
creat_agent,tool,mcp -> 可以查看 src/ui_test_demo

学习ReAct模式可以看下src/ReAct_demo

graph -> 可以查看 src/graphs_examples

rag相关: 
    多模态,文档解析: 先看src/file_agent
    稍微整合一下,工程化,看src/file_rag
    对于嵌入模型,嵌入向量,及文档检索加上agent -> src/agentic_rag

将rag做成一个小的项目,可以看下easy_rag_server

之后继续优化,我们采用mcp + agent的模型
    可以看下src/rag_chat 和src/mcp_server_rag

一个简易的rag后台管理系统+前端+对话系统 ，后续应该还会有v2的版本
